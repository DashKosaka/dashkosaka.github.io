OpenRadar: Pioneering Open-Source mmWave Radar Processing

In October 2019, the PreSense team launched OpenRadar, a groundbreaking project supported by the IBM C3SR Alchemy program. OpenRadar is the first open-source library that provides a generalized approach to processing mmWave radar data, helping researchers around the world unlock new possibilities in radar technology.

What is OpenRadar?

OpenRadar offers a complete signal processing pipeline for mmWave radar, capable of transforming raw radar data—such as range, Doppler, azimuth, and elevation—into 4D point clouds. This enriched data can be used for a variety of applications, from robotics to autonomous vehicles. The project is utilized by hundreds of researchers globally, contributing to advancements in radar-based perception systems.

Key Features of OpenRadar

  • Signal Processing Pipeline: Converts raw radar signals into range, Doppler, azimuth, and elevation data, providing 4D point clouds.
  • Extensive Module Management: Handles key processes such as data transfers over UDP, Constant False Alarm Rate (CFAR) filtering, data formatting, machine learning (ML) integration, and tracker architecture optimization.
  • Educational Resources: The project includes a Jupyter Notebook educational series that explains radar concepts and provides tutorials for users to learn radar signal processing hands-on.
  • Machine Learning and Sensor Fusion: OpenRadar integrates applied ML algorithms and sensor fusion techniques to enrich radar data output and boost performance in real-world applications.
  • Support for Production Radars: The platform is compatible with Texas Instruments' mmWave radars, ensuring it can be used in production environments.

OpenRadar is powered by technologies like Python, C/C++, mmWave Radar, PyTorch, Kalman filtering, tracking, networking, and Jupyter Notebook, making it a versatile tool for radar-based research.

For anyone interested in exploring or contributing to the future of radar signal processing, OpenRadar offers a comprehensive and accessible platform to start with!
