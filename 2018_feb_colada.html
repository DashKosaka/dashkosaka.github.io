Colada - Neural Network Drink Generator

Summary

In February 2018, Colada was my first-ever hackathon project, created during HackIllinois 2018. The project aimed to develop a neural network capable of randomly generating drink recipes by analyzing a vast combination of potential ingredients. The project utilized a data crawling algorithm to scrape public websites for existing drink recipes, which were then used as training data for the neural network.

Key Contributions

  • Neural Network: Developed a neural network to generate random drink recipes from hundreds of ingredients.
  • Data Crawling: Implemented an algorithm to scrape drink recipes from public websites for use as training data.

Technologies Used

  • Python
  • Keras
  • Numpy
  • Pandas
