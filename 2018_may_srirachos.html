SrirachOS Operating System

Summary

In May 2018, my team and I developed SrirachOS, a command-line interface (CLI) based operating system as part of a group project. We entered this project into a Microsoft-supported design competition, where we earned a top 4 placement out of all submissions. The project required a significant amount of time for both development and debugging of complex technical problems, pushing us to implement high-level coding practices and maintain strong communication across the team.

Competition placement of SrirachOS

Key Contributions

  • File System Design: I was responsible for establishing the file system design and implementing pagination, ensuring smooth data management within the OS.
  • Multi-process Context Switching: I architected the multi-process context switching functionality, allowing for efficient handling of multiple tasks simultaneously.
  • Advanced Features: Developed a full filesystem keyword search, autocomplete, and command history to enhance the user experience.

Early stage functionality of system commands and shell

Challenges

The most significant challenge was overcoming the complexity involved in developing an operating system from the ground up. We maintained good coding practices, including thorough comments, version control, and continuous communication to ensure progress.
