Embedded Systems & Front-End Development at Cohesive Manufacturing

In September 2019, I took on a short-term position as a software engineering contractor at Cohesive Manufacturing, a startup focused on providing monitoring solutions for critical infrastructure in the manufacturing industry. My role spanned both embedded systems and front-end development, allowing me to work on cutting-edge projects that improved the reliability of critical equipment monitoring while enhancing user experience through web applications.

Key Contributions

  • Cloud-Monitoring Engine: Developed the core embedded systems for a cloud-based engine to monitor and control mission-critical equipment in manufacturing environments.
  • Sensor Driver Development: Designed C++ software drivers for acquiring and interpreting ADC data from three different sensors, ensuring precise monitoring and data reliability.
  • Real-Time Data Logging: Managed an Arduino-based embedded system to log structural data in real-time, achieving less than 20 ms latency for seamless cloud integration.
  • Front-End Enhancements: Improved the front-end interfaces of customer websites by adding multi-language support, which boosted user engagement and usability across different regions.

Key Project: Cloud-Monitoring Engine for Critical Equipment

A major aspect of my work at Cohesive Manufacturing was centered around developing a cloud-monitoring engine. This engine was designed to log, control, and react to irregularities in critical equipment across small and medium manufacturing facilities. By building and optimizing sensor drivers, I ensured that the system could acquire and interpret data from three mission-critical sensors, providing continuous monitoring to prevent operational losses.

  • Data Acquisition: I developed C++ drivers for sensors that measured various equipment parameters and processed ADC data, ensuring high precision in detecting changes in equipment status.
  • Real-Time Logging: With the integration of an Arduino system, I set up a real-time data pipeline to stream sensor data to the cloud with less than 20 ms latency, enabling prompt alerts and actions when irregularities were detected.

Key Project: Manufacturing Transactions Engine

In addition to the monitoring system, I also contributed to the development of a manufacturing transactions engine, which powers web applications designed for end-customers and supply chain engineers. This engine was created to streamline the execution of manufacturing transactions in complex supply chain arrangements, including large OEMs and job shops across the country.

  • User Interface: The application features a modern 3D browser visualization tool, allowing users to interact with and manage manufacturing jobs easily.
  • Multi-Language Support: I enhanced the user experience by enabling multi-language support across customer-facing web applications, which broadened the usability of the application for diverse customers.

Challenges

Working on both the embedded and front-end aspects of the project required balancing real-time data processing with user-friendly web interfaces. Ensuring that sensor data was both accurate and delivered promptly to the cloud while simultaneously optimizing front-end performance for global users posed a unique challenge. The combination of these technologies required careful resource management, both in terms of computational efficiency on the embedded side and ease of use for the end customers.

Technologies Used

  • Embedded Systems: Arduino, C++, Microcontrollers
  • Front-End Development: HTML, Javascript, Multi-Language Support
  • Networking: Cloud Integration, Data Logging
  • Platforms: Ubuntu, Web Applications

This experience showcased the challenges and rewards of working on mission-critical systems that require both technical precision and user-focused design. My time at Cohesive Manufacturing strengthened my expertise in embedded systems and front-end development, providing valuable insights into real-time data management and the critical role of responsive interfaces in industrial applications.
